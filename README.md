- 의류수거함 검색의 불편함을 개선하기 위해 개발한 iOS 어플리케이션입니다. 
- 지도에서 내 위치 근처의 의류수거함을 찾을 수 있고, 지역구 별 의류수거함 현황을 확인할 수 있습니다. 

- 기술 스택 
  - 개발 환경
    - iOS: swift5, xcode 14.2
  - 라이브러리
    - KakaoMap API (REST API, Maps API)
    - Deployment Target : iOS 15.3
